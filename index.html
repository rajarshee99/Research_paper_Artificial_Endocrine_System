Artificial Endocrine System

Bio-Inspired Adaptive Control Framework – Research Paper

Abstract

The Artificial Endocrine System (AES) is a bio-inspired computational framework designed to model endocrine regulation mechanisms for adaptive, time-dependent, and distributed control in intelligent systems. By using hormone-like signals, AES enables smooth adaptation, contextual modulation, and long-term stability in LLM.
